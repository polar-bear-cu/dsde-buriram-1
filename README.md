## OCR Result Format

- District (เขต)

```
{
  "summary": {
    "จำนวนบัตรทั้งหมด": 0,
    "บัตรดี": 0,
    "บัตรเสีย": 0,
    "ไม่เลือกผู้ใด": 0
  },
  "data": [
    {
        "ชื่อผู้สมัคร": "",
        "พรรค": "",
        "คะแนน": 0
    },
    ...
  ]
}
```

- Partylist (บัญชีรายชื่อ)

```
{
  "summary": {
    "จำนวนบัตรทั้งหมด": ...,
    "บัตรดี": ...,
    "บัตรเสีย": ...,
    "ไม่เลือกผู้ใด": ...
  },
  "data": [
    {
      "พรรค": "",
      "คะแนน": ...
    },
    ...
  ]
}
```

- Referendum (ประชามติ)

```
{
  "summary": {
    "ผู้มีสิทธิ": ...,
    "มาใช้สิทธิ": ...,
  },
  "data": [
    {
      "รายการ": "เห็นชอบ",
      "คะแนน": ...
    },
    {
      "รายการ": "ไม่เห็นชอบ",
      "คะแนน": ...
    },
    {
      "รายการ": "ไม่แสดงความคิดเห็น",
      "คะแนน": ...
    }
  ]
}
```
